# Frukt & Sånt — Landing Page

A bright, mobile-first landing page for [Frukt & Sånt](https://www.instagram.com/fruktosant/), Stockholm's TikTok-viral fresh-fruit cafe.

**Live:** https://fos.triadsolutions.se

## Stack

Plain HTML, CSS and a dash of JS — no build step. Deployed as a static site on Vercel.

## Local preview

```bash
# Just open index.html in a browser, or:
npx serve .
```

## Deploy

This project is configured to deploy automatically to Vercel on every push to `main`.
